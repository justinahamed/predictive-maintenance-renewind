# Predictive Maintenance Model (ReneWind)

## Overview
Developed classification models to predict wind turbine failures using sensor data.

## Problem
Unexpected equipment failures increase maintenance costs. The goal was to predict failures in advance.

## Approach
- Data preprocessing and feature engineering
- Addressed class imbalance using upsampling/downsampling
- Built classification models and tuned hyperparameters
- Evaluated model performance and optimized predictions

## Results
- Improved failure prediction capability
- Enabled proactive maintenance and reduced operational risk

## Tech Stack
Python, pandas, scikit-learn
